VirtuCards

Project Overview

VirtuCards is a series of game applications designed to meet the needs of card game aficionado with the safety precautions of the COVID-19 pandemic in mind. I worked in a team along with 5 other members:

My role in this project was to work on an assortment of tasks both front-end and back-end. I worked on establishing connection with the Photon servers; creating game rooms; designing UI for the cards; implementing the game logic for shuffling cards and implementing exit functionality. I worked with Unity, C#, Google Firebase and the Photon Library during this project. The tasks in this project were assigned to each member by choice as each member wanted to improve their own skills.

History

VirtuCards was created as a part of the group's CS 307: Software Engineering I class. We were tasked with spending the entire semester creating something worthwhile and given full control of the creative process as well. The only thing we were required to do is regularly update the undergraduate teaching assistants and the professor. After getting to know one another, our group, inspired by our remote lectures, decided to create a game that allowed people to bridge the distances created by the pandemic.

Back-end

With regards to the back-end functionality for this project, I implemented the code for connecting to our client and host to the Photon servers; logging in with Google; sound effects for in-game actions; shuffling the deck; and creating game rooms. To help us implement multiplayer functionality in our game, our group utilised the Photon PUN 2 package. This package made our work easier since the documentation and library contained functions that handled more complex things like connecting to Photon's master servers and creating lobbies and rooms.

Front-End

With regards to the front-end functionality for this project, I created some preset avatars for users to use and I implemented the code for them to choose one. Along with that, I added code that displayed all the preset avatars in a carousel and stored user's choice in their profile on Google Firebase. I also worked on code that allowed users to pick custom images to use as their avatars.

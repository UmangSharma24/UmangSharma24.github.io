VirtuCards - Unity Game for Desktop and Mobile platforms

VirtuCards is a series of game applications designed to meet the needs of card game aficionado with the safety precautions of the current global pandemic in mind. The game allows users to connect to play card games in a way that follows social distancing guidelines. My role involved working on establishing connection with Photon servers; creating game rooms; desiging UI for the cards; implementing the game logic for shuffling cards and implementing exit functionality. I worked with Unity, C#, Google Firebase and the Photon Library during this project.

Boilership 150 - Unity Game

Boilership is a Unity game built at the Hello World hackathon in 2019. The game is a platformer based on the Boilermaker culture - incorporating fun references and inside jokes from campus life. My role in this project was to create assets for the enemies in the game and the power-ups. I was responsible for implementing the back-end functionality for the power-ups as well. This was one of my first projects and taught me a lot about the process of conceiving and designing software. I was able to learn the basics of Unity and C# through my contribution to this project.

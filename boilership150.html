Boilership 150 - Unity Game



Boilership is a Unity game built at the Hello World hackathon in 2019. The game is a platformer based on the Boilermaker culture - incorporating fun references and inside jokes from campus life. My role in this project was to create assets for the enemies in the game and the power-ups. I was responsible for implementing the back-end functionality for the power-ups as well.

This was one of my first projects and taught me a lot about the process of conceiving and designing software. I was able to learn the basics of Unity and C# through my contribution to this project.

Repository: Here

Back-end work

For this Unity project, I implemented the functionality for the power-ups and I worked on the tracking algorithm for the enemy characters to identify the player character.

The power-up feature was implemented by attached a Tag field to the GameObject and assigning it a unique value. Upon contact with the power-up GameObject, the player character's Tag field is reassigned to the same value as that of the power-up. In the Update method of Unity (which is called once every frame), I then added an if statement that determines if the player character's Tag field matches that of the Power-up's. If the two are equal, the player character's speed value is doubled. After a short while, the tag value and the speed value of the player character are restored.
